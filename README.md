# 🤖 Chatbot Project

This is a simple AI chatbot built using Python and Streamlit.

## 🚀 Features
- Answers basic questions (fees, timing, contact)
- Simple and user-friendly interface
- Fast responses

## 🛠️ Tech Used
- Python
- Streamlit

## ▶️ How to Run
1. Install dependencies:
   pip install streamlit

2. Run the app:
   streamlit run chatbot.py
